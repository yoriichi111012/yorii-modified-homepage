# Yorii's Modified Homepage
## Welcome to Yorii's Modified Homepage Github
This is an extension which replaces the default homepage of chromium based browsers. 
It can currently only be installed through developer mode and loading it unpackaged.

## Screenshots
![screenshot](https://iili.io/fpOIQe9.md.png)

## Installation Instructions
1. Download the extension's zip and extract it.
2. Go to the browser's *extensions page* i.e *chrome://extensions*.
3. Enable *Developer Mode*.
4. Click *Load unpackaged* and then select the *extracted folder* of the extension.
